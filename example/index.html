aka-xe Example Redirect Page

This is an example of how a redirect page works on aka-xe.github.io.

Sample Redirect Code:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="refresh" content="0; url=https://example.com" />
    <title>Redirecting...</title>
  </head>
  <body>
    <p>Redirecting...</p>
  </body>
</html>
